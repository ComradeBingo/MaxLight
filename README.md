**Max Light** — это нативная Windows-обёртка для веб-мессенджера Max (web.max.ru), написанная на C# WinForms с использованием WebView2. Приложение добавляет отсутствующие в веб-версии возможности: системные уведомления, работу через трей, PIN-код, автозапуск и автоматические обновления через Velopack.

## ✨ Основные возможности

| Функция | Описание |
|---------|----------|
| **Веб-интеграция** | Встроенный браузер Microsoft Edge WebView2 с режимом InPrivate |
| **Уведомления** | Кастомные всплывающие окна с аватарками, инициалами и анимацией |
| **Сворачивание в трей** | Иконка в системном трее, смена иконки при непрочитанных сообщениях |
| **Автозапуск** | Опция запуска приложения вместе с Windows |
| **PIN-код** | Защита доступа к приложению при запуске |
| **Сохранение авторизации** | Токен и deviceId шифруются и хранятся в реестре Windows (DPAPI) |
| **Блокировка трекинга** | Фильтрация запросов к analytics, apptracer, sdk-api, adsystem |
| **Автообновления** | Дельта-обновления через Velopack (GitHub Releases) |

## 🔒 Безопасность

### Шифрование данных
- Все чувствительные данные (токен, deviceId, PIN-код) хранятся в зашифрованном виде в реестре Windows.
- Используется **DPAPI (Data Protection API)** с привязкой к текущему пользователю Windows.
- Алгоритм: AES-256 через Windows Crypto API.

### Защита от XSS
- Внедрение специальных скриптов защиты в WebView2.
- Санитизация ввода и CSP-заголовки.

### Блокировка трекера
Фильтруются HTTP-запросы, содержащие:
- `analytics`
- `apptracer`
- `perf/`, `sdk-api`
- `adsystem`, `crashtoken`
- `crash`, `track`

### PIN-код
- Проверка при каждом запуске.
- 3 попытки ввода до блокировки.
- Шифрованное хранение в реестре.

## 📦 Система обновлений (Velopack)

| Параметр | Значение |
|----------|----------|
| Источник | GitHub Releases |
| Тип обновлений | Дельта (скачиваются только изменённые файлы) |
| Проверка | При каждом запуске приложения |
| Установка | Фоновая, с перезапуском приложения |

### Файлы релиза
- `MaxLight-{version}-win-full.nupkg` — полный пакет
- `MaxLight-{version}-win-delta.nupkg` — дельта-обновление
- `MaxLight-win-Setup.exe` — установщик
- `releases.json` — манифест обновлений

## 📥 Установка и обновление

### Установка
1. Скачай `MaxLight-win-Setup.exe` из последнего релиза
2. Запусти установщик
3. Следуй инструкциям на экране

### Автообновление
При каждом запуске приложение проверяет GitHub на наличие новой версии и автоматически устанавливает её.

### Ручное обновление
Скачай и запусти новый `Setup.exe` поверх старой версии.

## ⚙️ Конфигурация и хранение данных

| Данные | Место хранения | Шифрование |
|--------|----------------|-------------|
| Токен авторизации | `HKCU\SOFTWARE\MaxLight\Auth` | ✅ DPAPI |
| DeviceId | Внутри зашифрованного AuthData | ✅ DPAPI |
| PIN-код | `HKCU\SOFTWARE\MaxLight\PIN` | ✅ DPAPI |
| Размер/позиция окна | `HKCU\SOFTWARE\MaxLight\WindowState` | ❌ Нет |
| Автозапуск | `HKCU\SOFTWARE\Microsoft\Windows\CurrentVersion\Run` | ❌ Нет |
